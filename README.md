# Отчёты по макроэкономическим данным

Скрипт читает CSV-файлы с показателями по странам (ВВП, безработица, население и т.д.) и выводит в консоль отчёты в виде таблиц.

## Запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Формирование отчёта «среднее ВВП по странам» по одному или нескольким файлам:

```bash
python main.py --files economic1.csv economic2.csv --report average-gdp
```

Параметры:

- `--files` — один или несколько путей к CSV-файлам (данные объединяются).
- `--report` — тип отчёта. Сейчас доступен только `average-gdp`.

Пример вывода (фрагмент):

```
+----------------+---------------+
| country        |   average_gdp |
+================+===============+
| United States  |      23923.7  |
+----------------+---------------+
| China          |      17810.3  |
+----------------+---------------+
| Japan          |       4467    |
+----------------+---------------+
```

```bash
pytest

```

## Добавление нового отчёта

1. В `reports.py` опишите функцию, которая принимает `rows: list[dict]` (строки CSV как словари) и возвращает `(headers, table_rows)` — заголовки и список строк для таблицы.
2. Зарегистрируйте её в словаре `REPORTS`: `REPORTS["имя-отчета"] = ваша_функция`.
3. Новый отчёт будет доступен по `--report имя-отчета`.
